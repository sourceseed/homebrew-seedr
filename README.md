### homebrew seedr

The homebrew tap for seedr.


------


### Install seedr


step 1: Add official tap

```bash
brew tap sourceseed/seedr
```

step 2: Install seedr:

```bash
brew install seedr
```

### Update seedr

step 1: update tap

```bash
brew update
```

step 2: update seedr

```bash
brew upgrade seedr
```

------

### Uninstall seedr

step 1: uninstall

```bash
brew uninstall seedr
```

step 2: untap official formula

```bash
brew untap sourceseed/seedr
```
